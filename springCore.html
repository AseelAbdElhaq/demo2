Spring Core

  1. Inversion of Control (IoC) Container
    A core component of Spring that controls object creation and automatically injects dependencies.
  2. Aspect-Oriented Programming (AOP) Framework
    Allows separation of cross-cutting concerns.
  3. Data Access Framework
    Simplifies interaction with databases.
  4. Transaction Management Framework
    Handles database transactions.
  5. Model-View-Controller (MVC) Web Framework
    Organizes application structure.
